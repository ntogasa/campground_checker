API

Other scrapers may benefit from access to a campground ID dataset. Hence, Campground Checker offers an API with public read permissions.

Each 'Campground' object in the database contains a campground name, campground ID number, and parent zone name.


If you would like to contribute any data, please reach out to me at ntogasa [at] gmail [dot] com
