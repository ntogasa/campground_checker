What is this?

Campground Scavenger is here to make it easier for you to find campground availabilities.

If you've reserved a campground online before, chances are you've used recreation.gov to do so. A huge inefficiency (that I believe is intentional design) of recreation.gov's reservation interface tis that you can only check one campground at a time for availabilities.

This means that if you want to know if ANY campsites are available in Yosemite National Park, you need to visit the web page of each campsite to find out.

Campground Scavenger on the other hand, let's you mass search as many campgrounds as you want. At the click of a button, you can search every reservable campground in Kings Canyon National Park!

How to use
  1. go to Campgrounds and search your desired parent zone (i.e. Kings Canyon National Park)
  2. copy the ID numbers, and go to the Availability page
  3. paste them in the input field, along with your target dates, and submit
  4. wait for your results to appear!

I hope this makes it easier for you to plan your next outdoor excursion. Enjoy!


API

You can visit the API to access our campground database. Each campground object in the database includes a name, ID number, and parent zone name.
